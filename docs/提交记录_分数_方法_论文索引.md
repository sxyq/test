## 6. 后续策略

1. 短期线上最高可信包仍是 `V42-SILENT-RUNNER-sxyq.zip`。
2. 若继续线上实测，优先考虑 `V101-SINGLE-USER-CAUSAL-SDPA-sxyq.zip`，但它不是已证明超过 V42 的包。
3. 不建议继续围绕 `V84/V99` no-pred/H2D skip 家族做小调参，线上结果已经低于 V42。
4. 真正高上限方向应转向真实 cached-shard/A800 证据驱动的 sparse layout、attention backend 或 SMoE backend，而不是继续提交本地 synthetic-only probe。
